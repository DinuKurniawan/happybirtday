# 🎨 Features & Technologies

## ✨ Main Features

### 1. **Birthday Landing Page**
- Beautiful gradient background (purple/blue theme)
- Glassmorphism card design
- Fully customizable content
- Auto-updating date display
- Responsive design for all devices

### 2. **Admin Panel**
- Clean and intuitive interface  
- Real-time form validation
- Secure access via URL parameter or keyboard shortcut
- Easy content management

### 3. **Content Management**
- ✏️ Customizable recipient name
- 📅 Date picker for birthday
- 🎵 Music file upload (MP3, WAV, OGG, etc.)
- 🖼️ **UNLIMITED Image Upload** - Upload as many photos as you want!
  - Multiple selection support (Ctrl+Click or Shift+Click)
  - Live preview with delete button
  - Grid layout with responsive design
- 💬 Custom birthday message
- 💾 LocalStorage persistence

### 4. **Animations & Effects**

#### **Confetti Effect**
- Celebratory confetti animation on page load
- Automatically stops after 10 seconds
- Responsive to window resize

#### **Floating Icons Animation**
- Background emoji decorations (🎈🎁🎉🎂)
- Smooth up-and-down floating motion
- Low opacity for non-intrusive effect

#### **Pulse Animation**
- Icon heartbeat effect
- Continuous gentle scaling
- Applied to cake, gift, and heart icons

#### **Fade & Slide In**
- Staggered entrance animations
- Content appears from bottom to top
- Spring-based smooth transitions

#### **Hover Effects**
- Button scale on hover
- Interactive feedback
- Smooth transitions

#### **Balloon Animation**
- Bottom emoji row with continuous motion
- Gentle floating effect

#### **Image Gallery Animation**
- Staggered fade-in for each image
- Scale animation on hover
- Smooth spring transitions
- Responsive grid layout
- Photo counter badge on each image

### 5. **Music System**
- Auto-play on page load
- Loop playback
- Supports all common audio formats
- File upload with validation
- Blob URL storage

## 🛠️ Technologies Used

### **Core**
- **React 19** - Latest React version with improved performance
- **TypeScript 6** - Type-safe code
- **Vite 8** - Lightning-fast build tool

### **Animation Libraries**
- **Framer Motion 12** - Production-ready animation library
  - Variants system for complex animations
  - Spring physics
  - Gesture support
  
- **React Confetti 6** - Realistic confetti effect
  - Canvas-based rendering
  - Responsive dimensions
  - Customizable particles

### **UI Components**
- **React Icons 5** - Popular icon library
  - Font Awesome icons
  - Tree-shakeable imports
  - 40,000+ icons

### **Styling**
- **CSS-in-JS** (inline styles)
- **Linear Gradients**
- **Box Shadows**
- **Border Radius**
- **Glassmorphism effects**

### **Browser APIs**
- **LocalStorage** - Data persistence
- **File API** - Music upload handling
- **Blob URLs** - Audio file storage
- **URL API** - Query parameter handling
- **Audio API** - Music playback

## 📱 Browser Support

- ✅ Chrome/Edge (latest)

- ✅ Safari (latest)

## 🎯 Performance

- **Fast Initial Load** - Vite's optimized bundling
- **Code Splitting** - Automatic chunk optimization
- **Tree Shaking** - Remove unused code
- **Asset Optimization** - Compressed CSS/JS
- **Lazy Loading** - Components loaded on demand

## 🔒 Security

- No external API calls
- No data sent to servers
- All data stored locally in browser
- No cookies or tracking
- Privacy-focused design
